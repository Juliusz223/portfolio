Moje Projekty

  • Gra w C# - Zgadywanie liczby: To prosta gra, w której komputer wybiera losową liczbę, a gracz musi ją odgadnąć. Użyłem C# oraz Visual Studio. Po odgadnięciu liczby program pyta, czy chcesz zagrać jeszcze raz.
  • Gra w C# - Kamień, Papier, Nożyce: To klasyczna gra, w której gracz wybiera jeden z trzech ruchów, a komputer losowo wybiera jeden z tych ruchów. Użyłem C# oraz Visual Studio w Console App .NET Framework.
  • Ala Sklep: To aplikacja, która pozwala na przeglądanie różnych produktów technologicznych. Użyłem C# oraz Visual Studio w Console App .NET Framework do stworzenia logiki i zarządzania danymi produktów.
